Tailscale Watchdog

An Android foreground-service watchdog that reconnects Tailscale when the VPN drops. Built for OEMs that aggressively kill background services — Realme/ColorOS, Xiaomi/MIUI, Oppo, Vivo — where Android's built-in "Always-on VPN" gets overruled and the device becomes unreachable.

What it does

  • Runs a persistent foreground service with an ongoing notification (harder for the OS to kill).
  • Listens for VPN drops via ConnectivityManager.NetworkCallback.
  • Triggers Tailscale to reconnect via broadcast intent + activity launch.
  • Auto-starts on boot and on package replace.
  • Optional AccessibilityService fallback taps the Connect button as a last resort.

Requirements

  • Android 8.0+ (API 26)
  • Tailscale app installed (com.tailscale.ipn)

Build

./gradlew assembleDebug
adb install -r app/build/outputs/apk/debug/app-debug.apk

Or open the project in Android Studio (Iguana or newer).

The OEM keepalive problem

On aggressive ROMs the app alone isn't enough — you also need to set the app to Unrestricted battery, enable Auto-launch / Startup Manager, lock it in Recents, and turn on Always-on VPN + Block connections without VPN. The full ColorOS/MIUI checklist and architecture are documented in the README on GitHub.
